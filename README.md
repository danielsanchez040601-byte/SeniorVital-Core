## 📖 Descripción del Proyecto
**SeniorVital** es un sistema inteligente de gestión wellness diseñado exclusivamente para adultos mayores (+60). Utilizando un enfoque de **AI-Augmented Development**, la plataforma integra Agentes Autónomos (Wellness Coach y Preventivo) para generar rutinas de bajo impacto seguras, monitorear el progreso y ofrecer un "Modo Cuidador" para familiares, operando bajo una arquitectura *Offline-First* en Google Cloud Platform (GCP).
